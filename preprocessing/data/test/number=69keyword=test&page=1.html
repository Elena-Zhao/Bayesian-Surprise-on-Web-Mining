Computer-Administered (CAT) Tests

Most CAT tests are offered on a continuous basis. For these tests, you can determine specific test-date availability during the registration process. Other CAT tests are offered only during limited administration windows.

See the 2014-15 Limited-Administration CAT Testing Schedule (PDF) for administration windows, registration periods and score reporting dates for the 2014-15 testing year.

See also the list of all 2014-15 Texas Educator Certification Program tests (PDF). This list shows each test name, test code, question types and test dates.
